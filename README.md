---

## Quick Start

1. **Clone the repository**
```bash
git clone <repo-url>
cd thoughtforge
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the server**
```bash
npm run dev
```
(or use `nodemon index.js` directly if you prefer)

4. **Open in browser**
```bash
http://localhost:3000
```
