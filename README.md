## Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/Misba0019/ThoughtForge.git
cd ThoughtForge
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the server**
```bash
npm run dev
```
(or use `nodemon index.js` directly if you prefer)

4. **Open in browser**
```bash
http://localhost:3000
```
